# ANGLICISMS_AND_CALQUES - Anglicismy a syntaktické kalky

**Verze:** 1.0  
**Účel:** Identifikace a nahrazení nevhodných anglicismů a doslovně přeložených anglických konstrukcí

---

## 🎯 PRINCIP

Kvalitní překlad by měl znít, jako by byl původně napsán česky. Anglicismy a kalky prozrazují, že jde o překlad, a narušují plynulost čtení.

## KATEGORIE A: LEXIKÁLNÍ ANGLICISMY

| ❌ Anglicismus | ✅ Česky | Poznámka |
|----------------|----------|----------|
| implementovat | zavést, uskutečnit, realizovat | |
| fokusovat (se) | soustředit se, zaměřit se | |
| adresovat (problém) | řešit, zabývat se | |
| finalizovat | dokončit, uzavřít | |

| randomní | náhodný | |

| signifikantní | významný, výrazný | ⚠️ pokud ne statisticky |
| relevantní | podstatný, důležitý, související | |
| koncept | pojem, myšlenka, návrh | |
| eventuelně | případně, možná, popřípadě | |
| prezentovat | představit, ukázat | |
| realizovat | uskutečnit, provést | |
| kompletní | úplný, celý | |
| extrémní | krajní, mimořádný | |

| iniciovat | zahájit, začít | |

| targetovat | zaměřit se, cílit | |
| editovat | upravit | |

| refreshovat | obnovit, osvěžit | |

### Akceptovatelné anglicismy (zdomácnělé)

Některé anglicismy jsou již součástí češtiny:
- email, internet, software, hardware
- marketing, management, design
- stres, relax, boom
- tým, gól, sport

**Pravidlo:** Pokud je slovo v SSJČ nebo Akademickém slovníku, je akceptovatelné.

## KATEGORIE B: SYNTAKTICKÉ KALKY

### 1. Nadměrné pasivum

Angličtina používá pasivum častěji než čeština. V češtině preferujeme aktivní konstrukce.

| ❌ Pasivum (anglický vliv) | ✅ Aktivně (česky) |
|---------------------------|-------------------|
| Bylo zjištěno, že... | Zjistili, že... / Ukázalo se, že... |
| Je předpokládáno, že... | Předpokládá se, že... / Vědci předpokládají... |
| Fosilie byly nalezeny... | Fosilie nalezli... / Našli fosilie... |
| Výzkum byl proveden... | Výzkum provedli... / Vědci provedli výzkum... |
| Data byla analyzována... | Data analyzovali... / Rozbor dat ukázal... |
| Výsledky byly publikovány... | Výsledky publikovali... / Vyšla studie... |

### 2. Nominalizace (podstatná jména místo sloves)

Angličtina nominalizuje více než čeština. Čeština preferuje slovesa.

| ❌ Nominalizace | ✅ Slovesně |
|-----------------|------------|
| Provedení analýzy ukázalo... | Když analyzovali... / Analýza ukázala... |
| Po dokončení výzkumu... | Když dokončili výzkum... / Po výzkumu... |
| Během provádění experimentu... | Když prováděli experiment... |
| Využití této metody... | Když použili tuto metodu... |
| Při zkoumání dat... | Když zkoumali data... |
| Za účelem zjištění... | Aby zjistili... |

### 3. Anglický slovosled

Čeština má volnější slovosled, ale některé pozice jsou přirozenější.

| ❌ Anglický slovosled | ✅ Český slovosled |
|-----------------------|-------------------|
| V roce 2024 v Maroku vědci objevili... | Vědci v roce 2024 objevili v Maroku... |
| Často vědci zjišťují... | Vědci často zjišťují... |
| Pouze dva vzorky byly... | Byly pouze dva vzorky... / Jen dva vzorky... |
| Právě tento nález... | Tento nález právě... |

### 4. Rozvité přívlastky před substantivem

Angličtina hromadí přívlastky před podstatným jménem. Čeština je raději rozvíjí za ním.

| ❌ Anglicky (přeplněné) | ✅ Česky (přirozeně) |
|------------------------|---------------------|
| 773 000 let stará v Maroku nalezená čelist | Čelist nalezená v Maroku, stará 773 000 let |
| Dosud neznámý druh velkého masožravého dinosaura | Dosud neznámý druh dinosaura – velkého masožravce |
| Nově publikovaná mezinárodní studie | Mezinárodní studie, která právě vyšla |

### 5. Gerundiální konstrukce

Angličtina používá -ing formy, které čeština nemá.

| ❌ Doslovně | ✅ Česky |
|-------------|---------|
| Vědci strávili roky studujíc... | Vědci strávili roky studiem... / Léta studovali... |
| Analyzujíc data, zjistili... | Při analýze dat zjistili... / Když analyzovali data... |
| Výzkumníci pracující na projektu | Výzkumníci, kteří pracují na projektu |

## KATEGORIE C: VAZBY A FRÁZE

### Kalky předložkových vazeb

| ❌ Anglický kalk | ✅ Česky |
|------------------|---------|
| na konci dne | v konečném důsledku, nakonec |
| v termínech (in terms of) | z hlediska, pokud jde o |
| na druhou stranu | naproti tomu, oproti tomu |
| když přijde na | pokud jde o, co se týče |
| z mého pohledu | podle mě, podle mého názoru |
| v kontextu | v souvislosti s |
| dělat smysl | dávat smysl |

### Kalky frázových sloves

| ❌ Anglický kalk | ✅ Česky |
|------------------|---------|
| přijít s (come up with) | vymyslet, navrhnout |
| přijít na to (figure out) | přijít na to ✅ / zjistit |
| jít přes (go through) | projít, prostudovat |
| ukázat se (turn out) | ukázat se ✅ / vyjevit se |
| běžet do (run into) | narazit na |

## KATEGORIE D: TYPICKÉ ANGLICKÉ KONSTRUKCE

### „The fact that..."

| ❌ Anglicky | ✅ Česky |
|-------------|---------|
| Skutečnost, že fosilie byly nalezeny... | To, že fosilie nalezli... / Nález fosilií... |
| Fakt, že vědci publikovali... | Publikování studie... / To, že vědci publikovali... |

### „It is/was... that..."

| ❌ Anglicky | ✅ Česky |
|-------------|---------|
| Je to právě tato metoda, která... | Právě tato metoda... |
| Bylo to v roce 2024, kdy... | V roce 2024... |

### „What... is..."

| ❌ Anglicky | ✅ Česky |
|-------------|---------|
| Co je důležité je... | Důležité je... |
| Co vědci zjistili bylo... | Vědci zjistili... |

### Zdůrazňovací konstrukce

| ❌ Anglicky | ✅ Česky |
|-------------|---------|
| Je důležité poznamenat, že... | Je třeba dodat, že... / Důležité je, že... |
| Stojí za zmínku, že... | Za zmínku stojí... / Pozoruhodné je... |
| Mělo by být uvedeno, že... | Nutno dodat, že... |

## KATEGORIE E: VYCPÁVKOVÁ SLOVA

Anglické texty často obsahují „fillery", které čeština nepotřebuje:

| ❌ Zbytečné | ✅ Stručně |
|-------------|-----------|
| velmi unikátní | unikátní |
| naprosto nezbytný | nezbytný |
| zcela kompletní | úplný |
| v současné době | nyní / dnes |
| v podstatě | (vypustit) |
| v zásadě | (vypustit nebo přeformulovat) |
| jaksi | (vypustit) |
| tak nějak | (vypustit) |

## ✅ KONTROLNÍ POSTUP

1. **Lexikální kontrola:** Projdi text a označ anglicismy
2. **Syntaktická kontrola:** Hledej pasivum, nominalizace, nepřirozený slovosled
3. **Vazby:** Zkontroluj předložkové vazby a fráze
4. **Test přirozenosti:** Přečti větu nahlas – zní česky?

### Rychlý test:
> „Kdyby tohle řekl rodilý Čech, který neumí anglicky?"

Pokud ne → pravděpodobně anglicismus nebo kalk.

---

## ⚠️ POZOR NA HYPERKOREKCI

Ne všechny anglicismy jsou špatné. Některá slova:
- Jsou již zdomácnělá (team → tým)
- Nemají dobrý český ekvivalent (hardware)
- Jsou terminus technicus v oboru

**Pravidlo:** Nahrazuj jen tam, kde existuje přirozený český ekvivalent a kde anglicismus ruší.
